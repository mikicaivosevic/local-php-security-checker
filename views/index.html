PHP Security Checker

PHP Security Checker is a tool that checks if your PHP application depends on PHP packages with known security vulnerabilities. It uses the Security Advisories Database behind the scenes.



How to use ?

curl --data "@composer.lock" http://php-security.abstract.rs/api/v1/check


API response (example) 

{

      {
         "package_name":"laravel/framework",
